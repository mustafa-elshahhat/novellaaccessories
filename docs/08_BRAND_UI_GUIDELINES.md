# Brand & UI Guidelines — Novella Accessories

## 1. Reference Source

The brand direction is based on the supplied Novella reference images:

- A warm ivory square logo image.
- An Arabic announcement poster.
- An English announcement poster.

The visual identity is a soft luxury jewelry direction with delicate feminine details.

## 2. Brand Keywords

```text
soft luxury
feminine
warm
calm
premium
story-driven
delicate
elegant
minimal
jewelry-focused
```

## 3. Logo Direction

The logo reference uses:

- Lowercase `novella` wordmark.
- Elegant wide letter spacing.
- Monogram with the letter `N`.
- Feminine profile line art.
- Jewelry pendant detail.
- Leaf detail.
- Fine circular framing.

Tagline:

```text
Jewelry That Tells Your Story
```

The UI should not distort the logo. Keep enough whitespace around it.

## 4. Suggested Color Palette

The palette should be derived from the reference images.

```text
Warm Ivory Background:      #F4E8DA
Soft Cream:                 #FFF7EF
Champagne Gold:             #C79A72
Rose Gold:                  #B98563
Bronze Accent:              #9C6B4F
Mocha Brown Text:           #5E4439
Soft Taupe Text:            #8A7368
Border Gold:                #D7B08A
Light Shadow Beige:         #E8D6C4
Deep Brown:                 #3F2D27
```

Usage:

- Backgrounds: warm ivory, soft cream.
- Primary accents: champagne gold and rose gold.
- Text: mocha brown/deep brown.
- Borders: thin champagne-gold lines.
- Buttons: rose gold or bronze with ivory text.
- Secondary buttons: ivory with gold border.

## 5. Typography Direction

The reference uses:

- Elegant serif wordmark.
- Clean readable body typography.
- Script/calligraphy accent for emotional headings.

Suggested direction:

- English headings: elegant serif or high-fashion serif.
- English body: clean readable sans-serif.
- Arabic headings: refined Arabic display font.
- Arabic body: readable Arabic sans-serif.

Do not use overly playful fonts.

## 6. Layout Style

The layout should use:

- Large whitespace.
- Centered hero compositions.
- Soft rounded cards.
- Thin decorative separators.
- Fine border lines.
- Subtle shadows.
- Minimal visual clutter.

## 7. Decorative Elements

Allowed decorative elements:

- Thin gold lines.
- Small star/spark symbols.
- Small heart outline.
- Leaf motifs.
- Soft palm/leaf shadows.
- Fine circular jewelry-inspired shapes.

Avoid:

- Heavy gradients.
- Loud colors.
- Neon accents.
- Harsh black blocks.
- Overcrowded backgrounds.

## 8. Buttons

Primary button:

```text
Background: Rose Gold / Bronze
Text: Ivory
Shape: Soft rounded pill or rounded rectangle
```

Secondary button:

```text
Background: Transparent or Ivory
Border: Champagne Gold
Text: Bronze/Mocha
```

Button labels should be short and clear.

## 9. Cards

Product cards:

- Warm cream background.
- Soft rounded corners.
- Subtle border.
- Product image with generous padding.
- Price typography clear but not aggressive.
- Discount badge in rose-gold/champagne style.

## 10. Storefront Hero

Hero should support:

- Promotional product image.
- Elegant title.
- Short subtitle.
- CTA.
- Discount highlight.

Hero should not feel like a loud sales banner. It should feel premium and refined.

## 11. Admin UI

Admin can be more practical than the storefront, but should still use:

- Light warm background.
- Clean tables.
- Gold/bronze accents sparingly.
- Clear status badges.
- Minimal decoration.

## 12. Image Direction

Product photography should use:

- Warm neutral backgrounds.
- Soft shadows.
- Close-up jewelry detail.
- Ivory/beige surfaces.
- Rose-gold/champagne accents.
- Clean product focus.

## 13. Tone of Voice

English tone:

```text
Elegant, warm, concise, personal, premium.
```

Arabic tone:

```text
راقي، هادئ، بسيط، قريب من العميل، بدون مبالغة.
```

Do not use aggressive sales language.

## 14. Example Brand Messages

English:

```text
Every piece tells a story.
```

```text
Jewelry that adds a soft detail to your everyday look.
```

Arabic:

```text
كل قطعة بتحكي قصة.
```

```text
تفاصيل ناعمة تضيف لمستك الخاصة.
```

## 15. UI Consistency Rules

- Keep background warm and light.
- Avoid pure white where soft cream is more appropriate.
- Use gold accents only for emphasis.
- Use consistent spacing.
- Use consistent border radius.
- Keep mobile screens clean and direct.
- Do not show exact stock count to customers.
